== Description ==

This plugin creates a metabox on the post-editor page where the author/editor/admin can save multiple contributors for the posts.
Please note: This plugin allows this functionality only for post type: post.
 
== Installation ==
 
1. Upload the entire wp-contributers folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
